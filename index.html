정보

python IDE만 해도 종류가 굉장히 많다. 데이터베이스는 어떤가. 이때 각종 프로그램들의 장단점, 특징 등을 얻으려면 우리는 구글링을 해야한다.

프로그래밍 입문자

프로그램이 한번에 오류없이 적용되면 너무 고맙다. 설치 도중 오류는 개발을 시작하기도 전에 진을 뺀다. 특히 입문자의 경우, 알려줄 사람이 없어 점점 구글링 마스터가 된다.

시간

상황에 따라 추가적으로 버전 업/다운의 필요성이 생길 때, 우리는 해당 프로그램, 라이브러리의 파일 다운로드 경로를 탐색하는데 소모되는 시간이 매우 비효율적이다.
